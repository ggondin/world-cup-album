Como usar

Um guia rápido para usar o controle sem complicação.

  1. Marque cada figurinha: use “Sim” para as que você tem e “Não” para as que faltam.
  2. Controle repetidas: use os botões − e + em cada card.
  3. Copie sua lista: o botão “Copiar lista” gera um texto com “Não tenho” e “Repetidas”.
  4. Compare trocas: cole as repetidas de outra pessoa e veja só o que falta para você.
  5. Backup JSON: use para salvar o controle completo e restaurar depois.
